Quick Start

Up and running in 60 seconds.

Clone the repo, add your Gemini API key, and start the dev server. No backend required — everything runs in the browser.

1

Clone & install dependencies

git clone https://github.com/user/InterviewAI.git cd InterviewAI npm install
2

Add your Gemini API key

echo "VITE_GEMINI_API_KEY=your_key_here" > .env
3

Start the dev server

npm run dev
4

Open in browser

http://localhost:5173

Requirements: Node.js 18+, a Gemini API key from Google AI Studio, and a modern browser with WebSocket & WebGPU/WASM support.

Code Editor

Monaco, synced with the AI.

During live coding rounds, a full Monaco editor appears. Code context is debounced and injected into the AI's awareness — it can see what you write without interrupting.

💻 Live Sync

Code changes are debounced and silently injected into the AI context. The interviewer sees your code without you saying a word.

📋 Paste Detection

Pastes of 40+ characters are flagged as high-severity integrity events. The AI may casually check in after a paste.

🌐 Language Picker

Supports multiple languages with syntax highlighting. The language context is passed to the AI along with the code.

🛡 Integrity Tracking

Every paste event is logged with a timestamp, character count, and severity level in the proctoring timeline.

Integrity Monitor

Three layers of cheat detection.

On-device ML. Browser signals. AI vision. Working together to maintain interview integrity without compromising candidate experience.

Layer 1
Behavioral Signals
  • Fullscreen exit
  • Tab switch / minimize
  • Window blur
  • Paste detection (40+ chars)
  • Multi-monitor detection
Browser-event-based, instant detection
Layer 2
On-Device ML
  • No face detected
  • Multiple faces detected
  • Head pose tracking (yaw > 28°, pitch thresholds)
MediaPipe FaceLandmarker · GPU-accelerated WASM · ~4fps · 2.5s debounce
Layer 3
AI Vision Confirmation
  • JPEG frame sent to Gemini 2.5 Flash
  • Conservative confirmation only
  • Rate-limited with 120s backoff on 429
Triggered by soft ML flags (no-face, looking-away)
