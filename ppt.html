Architecture: Part 2

The Tri-Agent Architecture

Streaming 45 minutes of candidate video to a Cloud LLM is cost-prohibitive. We bypass this entirely using an asymmetrical Tri-Agent structure.

AGENT 01 · WEBSOCKET

The Conversationalist

Runs on Gemini Live. Handles natural speech and verbal probing based on the Orchestrator's active prompt.

Payload: Audio + Text only
SAVES 100% VIDEO API FEE
AGENT 02 · ON-DEVICE WASM

Frontend ML Watchdog

Uses MediaPipe running entirely inside the client's GPU. Tracks eye pitch, multiple faces, and copy-pastes locally.

THE "TRAP" MECHANISM:

If anomaly detected → sends lightweight JSON flag to Agent 1 to casually ask: "Everything okay?"

AGENT 03 · ASYNC BATCH

The Evaluator

Operates post-interview. Reads the raw text transcript to calculate a highly accurate, stage-by-stage scorecard.

High Accuracy · Cheap Batch Tokens
